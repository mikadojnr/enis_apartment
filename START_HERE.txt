1. Direct Unit Booking (2 min)
   → Go to /units → Click any unit
   → Select future dates in sidebar
   → Click "Check Availability"
   → See instant response with price
   → Click "Proceed to Booking"

2. Guest Booking (3 min)
   → Go to /bookings/new (without login)
   → Fill email, phone, first name, last name
   → Select dates and continue
   → Complete booking
   → Save guest code from confirmation
   → Visit /bookings/guest/[CODE] to test

3. Admin Booking (3 min)
   → Login as admin
   → Click "New Booking" on dashboard
   → Select "New Guest"
   → Fill guest details
   → Choose unit and dates
   → Click "Create Booking"

4. Profile Management (2 min)
   → Login as guest
   → Go to /auth/profile
   → Edit name/phone
   → Change password
