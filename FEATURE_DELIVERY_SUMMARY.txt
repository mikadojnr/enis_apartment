ADMIN WORKFLOW
================================================================================

Create Apartment Type:
  1. Go to /admin/apartments
  2. Click "+ New Apartment Type"
  3. Fill: Name, BR/BA, Area, Price
  4. Save

Create Unit:
  1. Go to /admin/units
  2. Click "+ New Unit"
  3. Select apartment type, set floor/view
  4. Add featured image URL
  5. Save → Redirects to gallery

Add Gallery Images:
  1. Click "+ Add Image"
  2. Paste image URL
  3. Add label (e.g., "Living Room")
  4. Add description
  5. Mark as featured if needed
  6. Save
  7. Repeat for each room

SUPPORT & TROUBLESHOOTING
================================================================================

Issue: Images not displaying
  Solution: Check URL is public and accessible
            Use HTTPS URLs
            Test URL in browser

Issue: Can't delete apartment
  Solution: Delete all units of that type first

Issue: Can't delete unit
  Solution: Cancel bookings for that unit first

Issue: Gallery not showing
  Solution: Add at least one image
            Verify image URLs are valid
            Clear browser cache

Issue: Featured image not updating
  Solution: Refresh page to see changes
            Clear cache if using same image

For more help: See GALLERY_MANAGEMENT.md troubleshooting section
